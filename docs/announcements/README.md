## Etiqueta da comunidade

- **Não cross-poste tudo no mesmo dia.** Posta no fórum primeiro (24-48h pra absorver feedback), depois Reddit, depois Discord.
- **Responde tudo nas primeiras 48h.** Showcase posts que somem após o launch viram desconfiança.
- **Não deletes feedback negativo.** Engaja honestamente — o app é v0.1, problemas são esperados.
- **Sem emojis em excesso, sem hype.** Comunidade Arch/Cachy não gosta de tom marketeiro.

## Antes de postar — checklist

- [ ] Repo público, README atualizado.
- [ ] AUR atualizada com a última tag (rodar `makepkg --printsrcinfo > .SRCINFO` e push).
- [ ] Pelo menos 1 screenshot (PNG, ≤500KB) ou GIF (≤2MB) na pasta `docs/img/` linkável.
- [ ] Issue tracker do GitHub habilitado e com `bug_report` / `feature_request` templates.
- [ ] Você está disponível nas próximas 48h pra responder.
